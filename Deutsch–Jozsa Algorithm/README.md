# Deutsch–Jozsa Algorithm ⚛️

## Overview
This project implements the Deutsch–Jozsa algorithm, which determines
whether a function is constant or balanced using a quantum circuit.

## Concepts Used
- Quantum superposition
- Oracle circuits
- Interference
- Quantum speedup

## Technologies
- Python
- Qiskit
- Qiskit Aer

## Outcome
The algorithm determines the function type in a single evaluation,
demonstrating quantum advantage.
