# Bernstein–Vazirani Algorithm ⚛️

## Overview
This project implements the Bernstein–Vazirani algorithm, which finds a hidden binary string using a quantum circuit with a single query.

## Concepts Used
- Quantum superposition
- Oracle circuits
- Interference
- Quantum speedup
- 

## Technologies
- Python
- Qiskit
- Qiskit Aer

## Outcome
The algorithm successfully identifies the hidden bitstring in one execution.
