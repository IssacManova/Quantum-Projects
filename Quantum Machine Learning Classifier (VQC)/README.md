## Overview
This project demonstrates a Variational Quantum Classifier (VQC) using Qiskit.
It combines quantum circuits with classical optimization to classify data.

## Concepts Used
- Variational Quantum Circuits
- Parameterized Gates
- Hybrid Quantum-Classical Training
- Feature Encoding

## Technologies
- Python
- Qiskit Machine Learning
- Scikit-learn
- NumPy

## Outcome
The quantum circuit learns to classify data using tunable quantum gate parameters.
