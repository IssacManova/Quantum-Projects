# Quantum Phase Estimation ⚛️

## Overview
This project demonstrates Quantum Phase Estimation (QPE), an algorithm used to estimate the phase of a unitary operator.

## Concepts Used
- Quantum superposition
- Controlled phase gates
- Inverse Quantum Fourier Transform
- Eigenvalue estimation

## Technologies
- Python
- Qiskit
- Qiskit Aer

## Outcome
The algorithm estimates the phase encoded in a quantum state using interference.
